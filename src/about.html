Purpose

Travel Explorer helps users discover locations by combining real-time images and mapping tools. The goal is to simplify destination research by showing pictures, maps, and essential information in one place.

Features

  • Search any destination worldwide
  • View high-quality photos powered by Unsplash
  • Interactive maps using Mapbox
  • Detailed destination page
  • Add and remove favorite locations
  • Favorites page with images and quick access
  • Responsive design for all screens

APIs & Technologies Used

  • Unsplash API (images)
  • Mapbox API (maps & coordinates)
  • JavaScript ES Modules
  • HTML + CSS
  • LocalStorage

Developer

This project was built by Eric Ayanru. The goal of the application is to explore modern web development concepts, API integration, modular JavaScript, and clean UI design.
